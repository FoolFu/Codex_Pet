# QQPetWidget

## 简介

这是一个把 QQ 宠物做成桌面挂件的 Electron 小工具。

现在这只企鹅不只是卖萌，还会和本机的 Codex 工作状态联动：

- 企鹅会根据 Codex 当前进度切换状态
- 头顶会显示当前状态文案
- 下方会显示一句气泡提示
- 气泡下面会显示 `Codex 额度` 面板
- 面板里会展示 `5h` 和 `7d` 两个额度窗口

适合一边写代码，一边在桌面右下角看 Codex 现在是在思考、读码、改码、执行，还是等你回复。

## 这个桌宠怎么玩

启动后你会在桌面右下角看到一只企鹅。

它的玩法很简单：

- 看状态：企鹅头顶会显示当前状态
- 看气泡：企鹅下方会用一句短文告诉你它现在在干嘛
- 看额度：展开 `Codex 额度` 可以查看 `5h` / `7d` 使用情况
- 点三角：点击 `Codex 额度` 右侧的小三角，可以折叠或展开面板
- 拖位置：按住企鹅本体区域拖动，可以把桌宠移动到你想放的位置

注意：

- 只有鼠标落在企鹅可交互区域时才能拖动
- `Codex 额度` 面板也支持鼠标交互，不会被点击穿透

## 企鹅会显示哪些状态

目前桌宠会根据 Codex 事件流切换这些状态：

- `待命`
- `思考`
- `读码`
- `改码`
- `执行`
- `等你`
- `完成`
- `出错`

你可以把它理解成一个挂在桌面的 Codex 小状态栏，只不过外形是一只 QQ 企鹅。

## 怎么启动

### 开发模式

先进入项目目录：

```bash
cd d:\mywork\stuff\ice-cream-headache.github.io-main\ice-cream-headache.github.io-main
```

安装依赖后启动：

```bash
npm install
npm start
```

常用命令：

```bash
npm start
npm run dev
npm run dist
```

其中：

- `npm start`：正常启动桌宠
- `npm run dev`：以开发模式启动，方便调试
- `npm run dist`：打包 Windows 安装包

### 安装包模式

已经打包好的安装包可以直接安装运行：

- `QQPetWidget-Setup-1.0.0.exe`

免安装版也可以直接双击：

- `win-unpacked/QQPetWidget.exe`

## Codex 联动是怎么工作的

这个桌宠会读取当前机器上的 Codex 本地数据目录，然后推断出桌宠状态和额度信息。

也就是说：

- 如果你的电脑上有 Codex 工作记录，企鹅会动起来并显示真实状态
- 如果没有对应的 Codex 数据，桌宠依然能启动，但通常只会显示待命或缓存状态

## 想让同事也能玩，需要什么

如果你把安装包发给同事：

- 同事可以正常安装并看到桌宠
- 但如果同事电脑上没有自己的 Codex 本地数据，状态联动不会完整生效

换句话说：

- 想看“企鹅桌宠”本身：安装包就够了
- 想看“Codex 状态联动”：目标机器上还需要有 Codex 运行产生的数据

## 当前交互说明

目前这个桌宠的交互重点是“轻量陪伴 + 状态可视化”：

- 企鹅本体显示在桌面上
- 状态标签挂在企鹅头顶
- 气泡和 `Codex 额度` 面板显示在企鹅下方
- 额度面板默认展开，可手动折叠

## 适合谁用

如果你符合下面这些情况，这个项目会很好玩：

- 你喜欢 QQ 宠物风格
- 你平时用 Codex 协助写代码
- 你想在桌面边角实时看到 Codex 当前正在干什么
- 你希望状态显示更轻松一点，而不是一直盯着终端或编辑器
